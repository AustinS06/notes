# Number row characters
* !: Exclamation Point, Bang
* @: Commercial at sign
* \#: number, hash or pound sign
* $: dollar sign
* %: percent sign
* ^: caret, hat
* &: ampersand, and-sign
* \*: asterisk, multiplication sign
* (: left parenthesis
* ): right parenthesis
